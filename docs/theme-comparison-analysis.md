# 周报主题对比分析

对比三个暗色主题：`dark-tech`、`dark-board`、`weekly-frost`（B方案）

---

## 色彩对比

| 元素 | dark-tech | dark-board | weekly-frost |
|------|-----------|------------|--------------|
| 背景 | `#0F172A` 深蓝 | `#0D1117` GitHub dark | `#0F1419` 深蓝灰 |
| 面板 | `#1E293B` | `#161B22` | `#1A2332` + `#1E2A3A` |
| 主色 | `#818CF8` 紫蓝 | `#58A6FF` 蓝 | `#38BDF8` 青蓝 |
| 文字 | `#E2E8F0` 冷白 | `#E6EDF3` 暖白 | `#E8ECF1` 冷白 |

---

## 核心差异化

### dark-tech — 技术文档风格

**定位：** API 文档、技术说明、单篇长文

**特点：**
- h1/h2 用 mono 字体（技术感强）
- 无面板色轮，简洁布局
- 适合连续阅读的长文档
- 无状态可视化设计

**不适合周报的原因：**
- 周报是多项目碎片化内容，需要状态区分
- 无本周/下周分界线概念
- 长文档排版不适合一周快速扫描

---

### dark-board — 仪表板风格

**定位：** 系统架构、项目看板、数据 dashboard

**特点：**
- 6色轮顶边（每个 section 不同颜色）
- 面板布局，视觉丰富
- 适合展示多个并行模块
- 认知负担高（颜色多，抢注意力）

**不适合周报的原因：**
- **色轮是噪音：** 周报项目状态本身需要颜色（进行中/阻塞/完成），面板再用色轮会互相抢
- **无时间线叙事：** dashboard 本质是"当前状态快照"，周报本质是"上周 → 下周连续性"
- **密度不足：** dashboard 面板间距大，周报需要紧凑呈现多个项目

---

### weekly-frost — 周报专用风格

**定位：** 工作周报、每周进展追踪

**独有设计：**

| 设计点 | 目的 | dark-tech | dark-board | weekly-frost |
|--------|------|-----------|------------|--------------|
| **本周/下周分界线** | 时间线叙事核心——上周结束、下周开始 | ❌ 无 | ❌ 无 | ✅ 灰色虚线 + "下周▸" 标签 |
| **时间线节点颜色区分** | 本周实色节点 vs 下周灰色节点（未来感） | ❌ 无 | ❌ 无 | ✅ primary 色节点 + 灰色未来节点 |
| **项目状态线（左 border）** | 用边框颜色区分状态，面板无色轮 | ❌ 无边框色 | ✅ 6色轮（噪音） | ✅ 状态线（进行中/阻塞/完成），面板无色轮 |
| **紧凑 KPI 网格** | 信息密度优先，4列紧凑 | 标准 3列 | 紧凑 2列（但无强调） | ✅ 4列紧凑 + delta 颜色 |
| **紧凑表格** | 减少 padding，一周内容多 | 标准 padding | 标准 padding | ✅ 紧凑 padding + mono 字体日期 |
| **唯一点缀色** | 减少视觉噪音，只让时间线和状态出声 | 紫蓝全局 | 6色轮 + 主色 | ✅ 只有时间线节点用 primary，其余安静 |

---

## 为什么 weekly-frost 是周报最优解

**周报的特殊性：**

1. **高频查看：** 每周都要看，必须"耐看"而非"好看"
   - dark-board 6色轮一周看一次觉得丰富，每天看会累
   - weekly-frost 只有青蓝点缀，长时间不刺眼

2. **碎片化内容：** 多个项目 + 日常 + 协作 + 下周计划
   - dark-tech 适合单篇长文，不适合 6-8 个 section 并行
   - weekly-frost 状态线让项目一眼可辨，不需要读标题

3. **时间线叙事：** "上周做了什么 → 下周要做什么" 连续性
   - dark-tech / dark-board 都没有本周/下周分界线
   - weekly-frost 分界线 + 灰色未来节点，时间连续性一目了然

4. **信息密度优先：** 周报内容多，每个像素都要有用
   - dark-board 面板间距大，浪费空间
   - weekly-frost 紧凑卡片 + 紧凑表格 + 紧凑 callout

---

## 决策

**差异化足够，值得创建新主题：**

- ✅ 独有本周/下周分界线（现有主题都无）
- ✅ 状态线替代色轮（减少噪音）
- ✅ 时间线节点颜色区分本周/下周
- ✅ 紧凑设计适配周报密度
