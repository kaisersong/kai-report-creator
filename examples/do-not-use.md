# Do Not Use

Do not use `kai-report-creator` for tasks that are not HTML report generation or review.

- Finished HTML to PPTX/PNG conversion: use `kai-html-export`.
- Slide decks or presentation creation: use `kai-slide-creator`.
- Raster image generation or editing: use the image generation workflow.
- General website/app building where the first screen should be an interactive product, not a report.
- Data analysis that requires inventing missing numbers; ask the user for data or keep placeholders.
